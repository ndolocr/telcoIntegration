# Telco Integration Platform

## Purpose

The Telco Integration Platform is a middleware application built using Spring Boot.

Its purpose is to provide a single, unified REST API that enables client applications to integrate with multiple mobile money providers without needing provider-specific implementations.

Initially, the platform will integrate with:

- Safaricom Daraja

Future integrations include:

- Airtel Money
- MTN MoMo
- Orange Money

---

## Objectives

- Learn Spring Boot from the ground up.
- Build using enterprise software engineering principles.
- Keep the system provider-independent.
- Use clean architecture and best practices.
- Document every architectural decision.
