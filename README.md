# Telco Integration Platform

A Spring Boot middleware application that provides a unified API for integrating with multiple mobile money providers.

## Current Status

🚧 Under Development

## Technology Stack

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Maven

## Planned Integrations

- Safaricom Daraja
- Airtel Money
- MTN MoMo

## Learning Goals

This project is being built as a practical journey to learn Spring Boot using enterprise software engineering practices.

Every architectural decision is documented inside the `docs/` directory.
